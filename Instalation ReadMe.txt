PETUNJUK INSTALASI

1. Jalankan dotnetfx35.exe untuk mengaktifkan .Net Services.
2. Jalankan CRRedist2008_x86.msi untuk mengaktifkan Cristal Report Services.
3. Copy File EXCEL.EXE di folder \Program Files\Microsoft Office\OFFICE11 ke \Program Files\Microsoft Office\OFFICE11\1033 dan rename menjadi xllex.dll.
4. Buatkan folder Program Files\CPI\PO Import sebagai directory aplikasi.
5. Copy/Replace semua file di folder "CPI" ke Program Files\CPI\PO Import di komputer user
6. Jalankan Program POIM.
7. Set database connection saat pertama aplikasi di jalankan :
	Nama Server / IP diisi 10.1.6.1 
	Nama DB di isi impr_dev
	Nama User di isi root
	Password di isi vbdev

8. Aplikasi siap di jalankan setelah terhubung ke database di server. Untuk memulai penggunakan masukkan user id dan password pengguna saat LOGIN.
